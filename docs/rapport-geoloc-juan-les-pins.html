Comment c'est cherché & la précision

✓ Ventes DVF (base du calcul) = PRÉCIS. Géocodage de l'adresse (BAN) → parcelle/sections cadastrales (IGN) → ventes réelles de l'État (DVF Etalab). Chaque vente a ses coordonnées exactes → pins bleus placés au bon endroit. C'est ça qui détermine la valeur.

≈ Annonces LeBonCoin = APPROXIMATIF. Recherche par commune (Antibes, rayon ~8 km) + surface ±30 %. LeBonCoin floute volontairement la position : on obtient le quartier (« Juan-les-Pins »), pas l'adresse → pins jaunes = zone, pas point exact.

À quoi ça sert & comment c'est présenté

  • DVF = socle de l'estimation : prix médian €/m² des ventes réelles comparables → valeur de marché.
  • Annonces = marché actif : ce qui est en vente maintenant (concurrence, prix affichés vs prix réels).
  • Présentation : carte interactive (pins + popups), KPIs €/m², tableau des annonces, fourchette de prix. Dans l'app : intégré à la fiche estimation + avis de valeur PDF.

Le géocodage utilise la Base Adresse Nationale (data.gouv) + cadastre IGN — sources officielles françaises, gratuites et précises. Google Maps n'est pas utilisé (on peut l'ajouter via une clé si tu veux des photos Street View / POI).
