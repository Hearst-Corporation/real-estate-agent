Chantiers structurants

1 · Auth hardening

  • MFA (TOTP/WebAuthn passkeys)
  • Révocation session Redis-backed + kid rotation JWT
  • OAuth2 (Google/MS) + audit log connexions
  • Évaluer Better Auth / Lucia v5

2 · Inngest + observabilité LLM

  • Sortir les appels DVF/ADEME du cycle HTTP (durable exec)
  • DLQ + retry/backoff natifs
  • Langfuse cost tracking par modèle/user
  • Structured output natif (SDK Anthropic 0.110+)

3 · Robustesse valorisation

  • Index INSEE/Notaires par département (vs national)
  • DPE rétro sur comps (loi Climat 2025+)
  • Outliers DBSCAN contextuels + micro-marché
  • ML hédonique (XGBoost) optionnel
